Key Insights

  • Academic performance dominates: Semester grades and approved units are the strongest predictors
  • Feature engineering helps: Custom features like approval_rate improved Decision Tree by 3.4%
  • Ensemble methods win: Random Forest and Gradient Boosting outperform simple classifiers
  • Class imbalance exists: "Enrolled" class (14% of data) is harder to predict accurately
  • Economic factors matter less: GDP, inflation show weak correlations with dropout
